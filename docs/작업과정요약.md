# PUBG 플레이어 행동 분석 웹사이트 구축 프로젝트 요약

## 📋 프로젝트 개요
- **목표**: PUBG 플레이어 행동 분석 결과를 시각화하는 웹사이트 구축
- **데이터**: 80만명 플레이어, 8개 클러스터, 99.25% 정확도 모델
- **기술스택**: HTML, CSS, JavaScript, Chart.js

## 🗂️ 프로젝트 구조

```
pubg-behavior-analysis/
├── 📁 css/              (CSS 스타일 파일)
├── 📁 js/               (JavaScript 기능 파일)  
├── 📁 assets/           (정적 자원)
├── 📁 data/             (분석 데이터 JSON 파일)
├── 📁 pages/            (서브 페이지들)
├── 📁 pubg_data/        (원본 분석 결과 파일들)
├── 📁 image/            (이미지 리소스)
├── 📁 docs/             (문서)
└── 📄 index.html        (메인 페이지)
```
