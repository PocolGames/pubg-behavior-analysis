## 🎨 PUBG 다크 테마 색상 팔레트 (Phase 3 예정)

### **PUBG 베이스 다크 컬러**
```css
:root {
  /* PUBG 다크 베이스 */
  --bg-primary: #0a0e1a;        /* 진한 네이비 블랙 */
  --bg-secondary: #1a1f2e;      /* 카드/패널 배경 */
  --bg-tertiary: #252b3d;       /* 호버/액티브 상태 */
  
  /* PUBG 게임 UI 색상 */
  --pubg-orange: #ff6b35;       /* 주요 액센트 (킬, 데미지) */
  --pubg-blue: #4a9eff;         /* 정보, 링크 */
  --pubg-green: #7ed321;        /* 성공, 생존 */
  --pubg-yellow: #f5a623;       /* 경고, 주의 */
  --pubg-red: #d0021b;          /* 위험, 데스 */
  
  /* 텍스트 */
  --text-primary: #ffffff;      /* 메인 텍스트 */
  --text-secondary: #b0b8c5;    /* 서브 텍스트 */
  --text-muted: #6c7284;        /* 비활성 텍스트 */
}
```

### **플레이어 유형별 다크 테마 색상**

  /* Survivor (생존형) - 그린 계열 */
